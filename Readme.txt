[TangleKai]_ReverseLayerOrder

Description:
--------------------
This script allows you to reverse the order of multiple layers at once in Photoshop.

Installation
--------------------
Move "[TangleKai]_ReverseLayerOrder_01.jsx" to "/Presets/Scripts/" in the Photoshop directory.

The next time Photoshop is opened, you can run the script from File->Scripts->[TangleKai]_ReverseLayerOrder.
